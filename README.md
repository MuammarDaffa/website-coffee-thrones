# Thrones Coffee

Thrones coffee adalah website yang dibuat menggunakan framework laravel yang bisa digunakan sebagai wadah promosi dari kedai kopi 

## Table of Contents

- [Fitur](#fitur)
- [Tim kami](#tim-kami)
- [UI](#ui)
- [Kritik dan Saran](#kritik-dan-saran)


## Fitur
### Admin
- Login dimana menggunakan akun admin 
- Halaman Dashboard : Admin ada kalimat selamat datang, bisa juga disebut landing page untuk admin
- Halaman Produk : menambahkan, mengedit dan menghapus dari kategori produk seperti kopi, nonkopi dan makanan. 
- Halaman Galeri : menambahkan, mengedit dan mengahapus galeri yang ditampilkan di halaman pelanggan
- Halaman Jam operasional : mengedit jam operasional yang akan berubah di halaman pelanggan  


### Pelanggan
- Halaman Beranda : landing page untuk pelanggan.
- Halaman Tentang : Sejarah berdirinya Thrones Coffee.
- Halaman Menu : Tampilan beberapa produk dari beberapa kategori seperti kopi, nonkopi dan makanan.
- Halaman Galeri : jepretan dari tempat Thrones Coffee
- Halaman Kontak : berisi sosial media seperti instagram, facebook dan tiktok
